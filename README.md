# Chat application.

A simple chat application project made by using React, Redux and websockets integrated.

## Demo link:
[https://hexlet-slack-project.herokuapp.com/](https://hexlet-slack-project.herokuapp.com/)

### Main features:

- realtime chat
- adding/deleting/renaming channels

### Built With

- AJAX
- Websockets
- React + Redux + Forms
- Bootstrap on React
- Webpack
